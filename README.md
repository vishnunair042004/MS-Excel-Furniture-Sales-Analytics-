## üìÇ Dataset Description

The dataset contains transactional-level furniture sales records with the following fields:

| Column                       | Description                                            |
| ---------------------------- | ------------------------------------------------------ |
| `product_category`           | Main product category (e.g., Dining Room, Outdoor)     |
| `product_subcategory`        | Subcategory such as Sofa, Bar Stool, Cabinet           |
| `brand`                      | Furniture brand                                        |
| `delivery_status`            | Delivery outcome (Delivered, Pending, Cancelled, etc.) |
| `assembly_service_requested` | Whether customer requested assembly                    |
| `payment_method`             | Payment mode used                                      |
| `order_id`                   | Unique order identifier                                |
| `customer_id`                | Unique customer identifier                             |
| `product_price`              | Product sale price                                     |
| `shipping_cost`              | Shipping charges                                       |
| `assembly_cost`              | Assembly service charges                               |
| `total_amount`               | Final amount charged                                   |
| `delivery_window_days`       | Estimated delivery time                                |
| `customer_rating`            | Customer satisfaction score                            |

This dataset is used to model real-world operational metrics in furniture retail.
